Here is the list of the main Matlab functions and sub-functions of MTT2col. Key steps are in bold, variables are in purple and comments are in green, for a brief description. This fully automated procedure allows to build trajectories from dual-color video acquisitions and to extract several descriptors, as explained in the main text.
Inputs, for each field of view: ¥ - Dual-color time-stacks (i.e. left: green / right: magenta)¥ - Dual-color ÔDICÕ image (transmitted light, in a subfolder named 'dic', optional but used for registration)¥ - Dual-color z-stacks (in a subfolder named 'z', optional)
Analysis:MTT_2_colors % MTT dedicated to 2 colorsMTT23i (method = 'coloc')detect_reconnex_23 => trajectories (Rouger et al., 2012; SergŽ et al., 2008)sort_traces % select long & fast enough tracesct4_by_file, ct5_by_file => histograms & statistics, including diffusion & trajectory descriptorsfit_directed_by_file => sort traces for D & ?cartobyf => maps for all filesreg_2_colors => compute registration from DIC images, save mean_tform.mattraj_xy_coloc => map with colocalizations, registering magenta tracescoloc3 => compute distance from green to magenta SMmerge_colors => magenta/green stacks & max. projectiondetect_cell_contact => statistics at cell contact onlycd(zdir) % analyze z-stack dataMTT23i => run MTT on z-stackgradient_z2 => z profile of z-stackmerge_colors => for z-stack
